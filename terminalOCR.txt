终端代码解压下载的推理模型.tar文件
tar -xvf ch_ppocr_server_v2.0_det_infer.tar

终端代码用推理模型对图片进行检测识别
通用模型：python tools/infer/predict_system.py --image_dir=D:/Dataset/LiquorBrand/ --det_model_dir=./inference/ch_ppocr_server_v2.0_det_infer/  --rec_model_dir=./inference/ch_ppocr_server_v2.0_rec_infer/ --cls_model_dir=./inference/ch_ppocr_mobile_v2.0_cls_infer/ --use_angle_cls=True --use_space_char=True --use_gpu=True
轻量模型：python tools/infer/predict_system.py --image_dir=D:/Dataset/LiquorBrand/ --det_model_dir=./inference/ch_ppocr_mobile_v2.0_det_infer/  --rec_model_dir=./inference/ch_ppocr_mobile_v2.0_rec_infer/ --cls_model_dir=./inference/ch_ppocr_mobile_v2.0_cls_infer/ --use_angle_cls=True --use_space_char=True --use_gpu=True
EAST文本检测模型推理：
首先将EAST文本检测训练过程中保存的模型，转换成inference model。以基于Resnet50_vd骨干网络，在ICDAR2015英文数据集训练的模型为例
python tools/export_model.py -c configs/det/det_r50_vd_east.yml -o Global.pretrained_model=./train_models/det_r50_vd_east_v2.0_train/best_accuracy Global.load_static_weights=False Global.save_inference_dir=./inference/det_east
EAST文本检测模型推理，需要设置参数--det_algorithm="EAST"，可以执行如下命令：
python tools/infer/predict_det.py --det_algorithm="EAST" --image_dir="D:/Dataset/LiquorBrand/" --det_model_dir="./inference/det_east/" --use_gpu=True

参数默认设置为limit_type='max', det_limit_side_len=960，可利用下面命令修改：
--det_limit_type=max --det_limit_side_len=3840

python tools/infer/predict_system.py --det_limit_type=max --det_limit_side_len=3840 --image_dir=C:/Users/www19/Desktop/1/ --det_model_dir=./inference/ch_ppocr_server_v2.0_det_infer/  --rec_model_dir=./inference/ch_ppocr_server_v2.0_rec_infer/ --cls_model_dir=./inference/ch_ppocr_mobile_v2.0_cls_infer/ --use_angle_cls=True --use_space_char=True --use_gpu=False
